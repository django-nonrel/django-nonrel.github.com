Django non-rel: Making Django run on non-relational databases

Django non-relational is a project to support Django on non-relational ('NoSQL') databases. Currently, we support MongoDB and Google's App Engine Datastore.

Code

All the code can be found on GitHub: https://github.com/django-nonrel.

Docs

The docs live here on the readthedocs, and you can submit pull requests here.

Pushing upstream
